GrowthKit Backend (Q1 MVP)
===========================

Этот каталог содержит серверную часть проекта для **Квартала 1 (MVP «Минимальное ядро»)**.

### Модули MVP

- `app/auth` — регистрация и вход через Telegram.
- `app/analytics` — базовая аналитика собственного канала (рост, ER, простые графики).
- `app/channel_search` — ручной поиск каналов в Telegram.
- `app/ai_content` — базовый AI‑генератор постов (до 5 генераций в неделю).

На этом этапе здесь только структура каталогов и документация без реализации бизнес‑логики.
