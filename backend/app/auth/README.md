Auth Module (Telegram Registration)
===================================

Задачи модуля:

- Привязка пользователя GrowthKit к Telegram‑аккаунту.
- Регистрация и вход через Telegram (метод Login Widget / OAuth‑подход).
- Управление базовыми сущностями пользователя и его подключёнными каналами.

В Q1 здесь будут только:

- Эндпоинты аутентификации.
- Простейшая модель пользователя и связка с одним каналом.

Сейчас модуль представлен только как каталог с описанием.
