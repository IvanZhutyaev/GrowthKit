Channel Search Module (Manual Discovery)
========================================

Задачи модуля:

- Ручной поиск каналов по ключевым словам и базовым фильтрам.
- Получение и отображение открытых метрик по найденным каналам.
- Подготовка фундамента для будущего AI‑поиска и AI Scout в Q4.

В Q1:

- Только ручной поиск и выдача результатов.
- Без автоматизированных рекомендаций и сложной разведки.

Сейчас модуль представлен только как каталог с описанием.
