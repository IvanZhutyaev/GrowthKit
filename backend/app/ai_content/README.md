AI Content Module (Post Generator)
==================================

Задачи модуля:

- Интеграция с AI‑провайдером (например, OpenAI GPT‑4o) для генерации постов.
- Поддержка базовых параметров промпта: тема, тон, длина, формат.
- Ограничение количества генераций в неделю согласно тарифу.

В Q1:

- Один основной сценарий: генерация поста для Telegram‑канала.
- Без сложных сценариев переработки контента и «умного сэндвича».

Сейчас модуль представлен только как каталог с описанием.
