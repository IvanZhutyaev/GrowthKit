Backend Application Structure (Q1 MVP)
======================================

Основные подсистемы серверного приложения:

- `auth` — авторизация/регистрация через Telegram, управление сессиями.
- `analytics` — агрегация и выдача базовой статистики по каналу (рост, ER).
- `channel_search` — ручной поиск каналов и выдача результатов.
- `ai_content` — обращение к AI‑сервису для генерации текстов постов.

Внутри каждого модуля будут:

- REST/HTTP‑эндпоинты (FastAPI).
- Сервисы доменной логики.
- Доступ к БД и внешним API (Telegram, AI).

Сейчас здесь только документация и структура, без кода эндпоинтов и моделей.
