Analytics Module (Basic Channel Metrics)
========================================

Задачи модуля:

- Сбор и хранение базовой статистики по каналу за 7+ дней.
- Предоставление данных для дашборда (рост подписчиков, ER, простые графики).
- Подготовка агрегатов для дальнейшего расширения аналитики в Q2+.

В Q1:

- Фокус на одном подключённом канале.
- Простые метрики и без сложного прогнозирования.

Сейчас модуль представлен только как каталог с описанием.
