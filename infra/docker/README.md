Docker Setup
============

План по Docker‑окружению:

- Отдельные Dockerfile для backend и frontend.
- Опциональные образы для PostgreSQL, Redis и вспомогательных сервисов в dev‑окружении.
- Базовая поддержка hot‑reload для локальной разработки.

Фактические Dockerfile будут добавлены после утверждения архитектуры.
