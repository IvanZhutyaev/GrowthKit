Infrastructure
==============

Этот каталог предназначен для инфраструктурных артефактов:

- Docker‑файлы для backend, frontend и вспомогательных сервисов.
- Конфигурация docker‑compose для локальной разработки.
- Манифесты Kubernetes для последующего развёртывания (Q2+).
- Конфигурация reverse‑proxy (Nginx) и CI/CD‑скрипты.

В Q1 важен упор на простой запуск локального окружения и подготовку к контейнеризации.
