Telegram Bot
============

Назначение:

- Канал коммуникации с пользователем: уведомления, триггеры, быстрые команды.
- Отправка ссылок‑deeplink в веб‑приложение и TMA.

В Q1:

- Минимальный функционал: базовые уведомления и тестовые команды.
- Умные триггеры и сложные сценарии будут добавляться в Q2+.
