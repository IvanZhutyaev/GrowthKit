UI Components Library
=====================

В этом каталоге будут размещаться переиспользуемые UI‑компоненты:

- Базовые элементы (кнопки, инпуты, модальные окна, карточки).
- Специализированные компоненты для графиков и метрик.
- Общие layout‑компоненты для дашборда.

В рамках Q1 важен быстрый и понятный UI для:

- Подключения канала.
- Просмотра базовых метрик.
- Запуска генерации постов и поиска каналов.
