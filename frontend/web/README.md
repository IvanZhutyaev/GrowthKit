Web Application (Next.js)
=========================

Основные зоны веб‑приложения для Q1:

- `pages` — маршруты Next.js (auth, dashboard, search, ai‑generator).
- `components` — переиспользуемые UI‑компоненты.
- `features` — функциональные модули (auth, dashboard, channel_search, ai_post_generator).

Задача веб‑части в Q1 — дать пользователю быстрый «Ага!-момент»:

- Подключить канал через Telegram.
- Увидеть базовые метрики.
- Попробовать AI‑генератор постов.

Код страниц и компонентов будет добавлен на следующих этапах.
