Dashboard Feature (Channel Overview)
====================================

Функциональность:

- Отображение базовых метрик канала (рост подписчиков, ER).
- Простые графики и ключевые показатели.
- Блок перехода к AI‑генератору и поиску каналов.

Модуль отвечает за первый «Ага!-момент» после регистрации.
