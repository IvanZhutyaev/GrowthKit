AI Post Generator Feature
=========================

Функциональность:

- Форма для ввода темы, цели и ограничений поста.
- Отправка запроса на backend AI‑модуль.
- Отображение сгенерированного текста и опций копирования/редактирования.

В Q1 реализуется базовый сценарий генерации до 5 постов в неделю.
