Auth Feature (Telegram Login)
=============================

Функциональность:

- Страница и компоненты для входа/регистрации через Telegram.
- Обработка состояния «подключён/не подключён».
- Перенаправление пользователя на дашборд после успешного входа.

Это каркас функционального модуля, без реализации логики авторизации.
