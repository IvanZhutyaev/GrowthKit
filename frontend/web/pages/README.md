Web Pages (Routing Layer)
=========================

Планируемые страницы для Q1:

- `/` — лендинг / экран приветствия.
- `/auth` — вход/регистрация через Telegram.
- `/dashboard` — дашборд базовой аналитики канала.
- `/channels/search` — ручной поиск каналов.
- `/ai/post-generator` — интерфейс AI‑генератора постов.

Файл содержит только план структуры маршрутов без реализации страниц.
