GrowthKit Frontend (Web + TMA)
===============================

Этот каталог содержит клиентскую часть продукта:

- Веб‑приложение (Next.js/React) — основной «командный центр».
- Telegram Mini App (TMA) — лёгкий интерфейс для быстрых действий с мобильного.

В Q1 основной фокус — веб‑дашборд и базовые экраны под MVP:

- Регистрация/вход через Telegram.
- Дашборд собственной аналитики (рост, ER).
- Экран ручного поиска каналов.
- Экран AI‑генератора постов.
