Telegram Mini App (TMA)
=======================

Назначение:

- Лёгкий интерфейс внутри Telegram для основных действий «на бегу».
- Быстрый просмотр ключевых метрик и уведомлений.
- Утверждение/отклонение сгенерированного AI‑контента.

В Q1:

- Можно ограничиться базовым экраном метрик и ссылками‑deeplink в веб‑приложение.
- Полное развитие TMA запланировано на последующие кварталы.
